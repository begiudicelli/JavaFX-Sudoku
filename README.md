# Sudoku em JavaFX

Projeto de Sudoku desenvolvido com JavaFX seguindo o padrão MVC, com foco principal na implementação do algoritmo de geração do tabuleiro.

## Destaque

O núcleo do projeto é um algoritmo de backtracking, responsável por:
- Gerar um tabuleiro completo válido
- Testar recursivamente valores possíveis
- Garantir regras do Sudoku (linha, coluna e bloco 3x3)

Após a geração, células são removidas com base na dificuldade:
- Fácil
- Médio
- Difícil

## Tecnologias

- Java
- JavaFX
- MVC

## Funcionalidades

- Geração automática de Sudoku válido
- Níveis de dificuldade
- Interface gráfica simples

## Observação

Projeto com foco educacional, destacando lógica de algoritmo e estruturação em camadas.
